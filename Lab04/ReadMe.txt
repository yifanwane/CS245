Description
A Java program to check if a string of parentheses is correctly matched. It uses a stack for validation.

Implementation
The program has two methods:

isReadable(String s): Checks if the input string contains correctly matched parentheses using a stack.

isValidPair(char open, char close): A private method to validate parentheses pairs.

Difficulties
Challenges include handling different parentheses types, addressing edge cases, and ensuring proper testing.
